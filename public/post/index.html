Caret vs. tidymodels - comparing the old and new

6 August 2019 in predictive modelling
In this post I will make a comparison between the most popular (by number of monthly downloads from Github) ML framework available for R to date: caret and its successor packages being written by the same author (Max Kuhn) that are wrapped together in a so called tidymodels framework. Tidymodels is a collection of different packages such as: rsample, recipes, parsnip, dials and more that allow for a running an ML project in a tidy format end-to-end.
